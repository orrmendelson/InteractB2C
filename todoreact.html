How to add interactions

×
  • (in Interactions list page, this page)
  • Login on upper-right corner, or create new user.
  • Open studio (the sand-box, see link on top of page)
  • (in studio)
  • Click the Plus button and "Snippet source"
  • Copy the code
  • (in Interactions list page, this page)
  • Go back to Interactions List page (this page)
  • Paste into the "New interaction source" field the copied text, and submit
  • The new interaction was added!
